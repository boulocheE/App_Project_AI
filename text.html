Architecture

Here, you are in the src/predict_from_scratch where I show from the very beginning how to predict. Here are other directories you might be interested in:

  • src/Data_augmentation: This directory is the one I used to augment data. I explain how & why to have the same amount of data for each category. I augmented twice the data. Make sure you check this notebook and this one. I also used this script to generate some zoomed images but it ended up being inacurate.
  • src/pannel_locator: This directory is focused on the shape classification. It's helpful in the filtering process to know where a sign is and if the shape is circular enough to run our sign model on it. Check this notebook to learn more about it.
  • src/Pannel_recognizer: This directory is the one I used to create a model. Check this script That I used to create our best_model.h5.

You can also find, in the Training directory what I trained my model on. Some images are about the gtrsb dataset. Most of the picture are from me: I screenshooted a lot of them on google street view. You can find some informations at the very bottom of this notebook or here.

Limitations

My road sign model is great (it has a val_accuracy > 0.90) but the issue (what I would have worked on if I had more time) is that the pannel_locator model isn't very accurate. it sometimes recognizes signs where there's no sign and is very sensitive to a red background or sign viewed from aside
